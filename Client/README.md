🌱 Green Gandhinagar

Green Gandhinagar is a modern eco-friendly waste pickup platform built with React. It provides a clean and responsive landing page experience with smooth animations, service highlights, FAQs, impact metrics, and user-friendly forms for signup and contact. The project focuses on promoting smart recycling and sustainable waste management in urban areas.

🚀 Features
✨ Animated Hero Section using Framer Motion
♻️ Eco-Friendly Waste Pickup Landing Page
📦 Service Cards with Interactive UI
❓ FAQ Section with Smooth Expand/Collapse
📞 Contact Form
📝 Signup Form with Local Storage Persistence
📊 Impact Metrics & Statistics
🔗 Smooth Anchor Navigation
📱 Fully Responsive Design
🎨 Modern and Clean UI/UX
🛠️ Tech Stack
⚛️ React.js
🎞️ Framer Motion
🎨 CSS / Tailwind CSS
🧠 JavaScript (ES6+)
💾 Local Storage

Install dependencies:

Run the development server:

npm run dev
